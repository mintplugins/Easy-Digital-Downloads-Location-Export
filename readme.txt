Export payment history by location and date. Useful for tax logs.

== Description ==

If you need the ability to export csv files based on date and country, most likely for tax reporting purposes, this allows you to create those files specific by country or by state.

Features of the plugin include:

* Export CSV files based on date and location

**Follow this plugin on [GitHub](https://github.com/easydigitaldownloads/Easy-Digital-Downloads)**

== Installation ==

1. Activate the plugin
2. Go to Downloads > Reports > Export
3. Select the date, country, and state you wish to export
4. Export the CSV
